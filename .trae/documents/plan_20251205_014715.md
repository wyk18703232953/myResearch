# 数据处理流程实现计划

## 1. 读取数据集
- 使用pandas读取mbpp_plus_data.parquet文件
- 提取每个数据条目的prompt、code、test和task_id字段

## 2. 实现LLM代码生成
- 基于spark-lite.py中的API调用方式，封装代码生成函数
- 确保生成的代码函数名与原code字段一致

## 3. 创建文件结构
- 为每个数据条目创建以task_id命名的子文件夹
- 在每个子文件夹中创建test.py文件，包含：
  - 生成的程序代码
  - 原数据条目中的test字段内容

## 4. 执行测试
- 遍历所有子文件夹，运行test.py文件
- 捕获执行结果，判断成功/失败

## 5. 生成报告
- 汇总测试结果
- 计算总数量、成功数量、失败数量和正确率
- 生成JSON格式的总报告

## 6. 实现步骤
1. 创建main.py主程序
2. 实现数据读取函数
3. 实现LLM代码生成函数
4. 实现文件创建和测试执行功能
5. 实现结果汇总和报告生成功能
6. 运行程序，执行完整流程

## 7. 代码结构
```
main.py
├── read_dataset()
├── generate_code()
├── create_test_file()
├── run_test()
├── generate_report()
└── main()
```
